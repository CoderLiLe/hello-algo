## 🚀 快速开始

### 环境要求
- Java 8+ 或 Python 3.6+
- Git

### 使用方式
```bash
# 克隆仓库
git clone https://github.com/CoderLiLe/hello-algo.git

# 查看特定算法
cd hello-algo/docs/动态规划
```

### 运行示例
```java
// Java 示例
javac Main.java
java Main
```

```python
# Python 示例
python3 sort/bubble_sort.py
```
